# MetroSystem - 上海地铁路径规划系统

基于C++的地铁路径规划系统，支持上海地铁网络的站点管理、最短时间路径规划和最少换乘路径规划。

## 功能特性

- **站点信息管理**：查询站点信息、线路站点展示、站点开关状态管理
- **最短时间路径规划**：基于Dijkstra算法计算两站点间最短通行时间路径
- **最少换乘路径规划**：计算两站点间换乘次数最少的路径
- **多路径选择**：支持输出多条备选路径（K=3）
- **站点状态管理**：支持批量/手动开关站点、关闭站点影响范围分析
- **路网连通性分析**：分析关闭站点对路网的连通性影响

## 数据说明

数据来源于上海地铁实际路网：
- `Data/Station.csv`：站点数据（站点ID、名称、所属线路、运营状态）
- `Data/Edge.csv`：边数据（起始站、终点站、线路、方向、通行时间）

包含线路：1号线~18号线、磁悬浮线、浦江线（共20条线路），约800+个站点。

## 编译与运行

### 环境要求
- Visual Studio 2022 (v143工具链)
- Windows SDK 10.0+
- C++17

### 编译步骤
1. 使用 Visual Studio 打开 `Metro/Metro.sln`
2. 选择 Debug/Release x64 配置
3. 生成解决方案

### 运行
编译后直接在 Visual Studio 中运行，程序会自动加载 `../Data/` 下的CSV数据文件。

## 算法说明

- **最短时间路径**：使用优先队列优化的Dijkstra算法，以边权重（通行时间）为度量
- **最少换乘路径**：使用双关键字Dijkstra算法，优先最小化换乘次数，其次最小化通行时间
- **K条最短路径**：通过逐条排除已有路径中的中间节点，重新运行Dijkstra来生成备选路径
